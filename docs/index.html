Uniswap V3 Exit & Swap Automation

This page describes a Gelato-powered service to:

  • Monitor a Uniswap V3 liquidity position on Base
  • Automatically exit the position when a price threshold is reached
  • Swap the withdrawn token into a stablecoin within a single transaction

Technologies used:

  • Solidity – custom smart contract
  • Gelato Network – automated task scheduling
  • Uniswap V3 – liquidity position management and token swap
